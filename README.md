# Django Rocketing

 A Django + React template to kickstart your new project.

 ## Using the template

 Tu use the template you have to have copier installed on your machine. Then:
 ```sh
 copier copy --trust "gh:kstypik/django-rocketing" /path/to/your/new/project
 ```
